Quick Start

Get Running in 5 Steps

1
Create Zoho OAuth credentials
Go to api-console.zoho.in → Self Client → add scopes above → generate refresh token. Duration: ~10 minutes.
2
Copy and fill .env
cp .env.example .env → paste ZOHO_CLIENT_ID, ZOHO_CLIENT_SECRET, ZOHO_REFRESH_TOKEN, and your ANTHROPIC_API_KEY.
3
Install dependencies
pip install claude-agent-sdk python-dotenv requests
4
Provision custom CRM fields (optional)
Add BB_Service_Type, BB_Destination, BB_Preferred_Room_Type, BB_Dietary, BB_Loyalty_Programs, BB_Travel_Style, BB_Budget_Tier fields via Zoho CRM settings to unlock the Butler Button concierge skills.
5
Run the agent
python agent.py → type "Give me my morning brief" to verify end-to-end connectivity.
