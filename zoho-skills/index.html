Setup

OAuth scopes — single Self Client at api-console.zoho.in

ScopeUsed by
ZohoCRM.modules.ALLAll agents — deals, leads, contacts, tasks, calls
ZohoMail.messages.ALL, ZohoMail.folders.ALLLead intake, deal won, pipeline velocity, follow-up drafter
ZohoCampaigns.campaigns.ALL, ZohoCampaigns.lists.ALLCampaign builder, personalization, automation builder, analytics
ZohoSocial.profiles.ALL, ZohoSocial.content.ALLSocial content engine
ZohoCliq.channels.ALL, ZohoCliq.messages.ALLAll agents — Cliq alerts
ZohoFlow.flows.ALLAutomation builder, flow inspection

Get running in 5 steps

1
Generate Zoho OAuth credentials
api-console.zoho.in → Self Client → paste all scopes above → generate refresh token (~10 min)
2
Fill .env
cp zoho-skills/.env.example zoho-skills/.env → add ZOHO_CLIENT_ID, ZOHO_CLIENT_SECRET, ZOHO_REFRESH_TOKEN, ANTHROPIC_API_KEY, ZOHO_MAIL_FROM
3
Install dependencies
pip install anthropic requests python-dotenv
4
Start webhook server + expose via ngrok
python zoho-skills/webhook_server.py in one terminal. ngrok http 8080 in another. Paste the ngrok URL into Zoho Flow webhook actions.
5
Run your first agent
python zoho-skills/agents/agent_morning_brief.py — if you get a brief in Cliq and email, everything is wired.
